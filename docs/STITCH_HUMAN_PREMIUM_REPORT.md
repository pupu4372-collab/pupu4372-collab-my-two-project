# Stitch Prompt: 지관재 Premium 평생 리포트 결과지

Design a premium Korean fortune report page for **K-Saju Pet**.

The page is a paid lifetime report result screen. It should feel like a handwritten letter or elegant reading paper, but with a deep night-sky mood. Use a calm, trustworthy, premium atmosphere rather than a flashy marketing style.

## Main Header

At the top, show the title:

**지관재(知觀齋)**

Under it, show this one-line maxim:

**지운자무애(知運者無礙) - 운명을 아는 자는 거침이 없나니.**

Below the maxim, include a short subtitle:

운명을 '아는 것(知)'에서 그치지 않고, 그 운명의 흐름을 멀리서 '관조(觀)'하며 대처하는 법을 익히는 서재와 같은 공간입니다.

## Visual Direction

- Overall mood: premium letter paper, quiet study room, deep night sky, soft mystic Korean philosophy.
- Background: layered dark navy, midnight purple, soft indigo gradients with subtle stars or dust-like light.
- Main report surface: warm ivory or moonlit cream letter-paper card.
- Border: strong and visible, darker than the current pastel cards. Use deep plum, midnight purple, or muted gold linework.
- Corners: rounded, but not too playful. Elegant rounded cards.
- Texture: subtle paper grain, faint constellation lines, or soft ink wash feel.
- Typography: large dignified Korean title, readable body text, generous line height.
- Avoid childish pet icons. This report is for a human premium reading.

## Layout

Create a mobile-first responsive layout.

1. Hero Letter Card
   - Large title: 지관재(知觀齋)
   - Maxim below
   - Small explanatory subtitle
   - Decorative top/bottom border inspired by old stationery or a seal stamp

2. Intro Letter Section
   - A warm greeting from 철학관장 심원
   - Make it look like a personal letter
   - Use quote-style typography or an inset paper block

3. Report Flow Sections
   The content should appear in this order:
   - 소개말
   - 서두
   - 만세력
   - 사주결과
     - 년사주
     - 년사주 총평
     - 평생사주
     - 평생사주 총평
   - 서비스 별자리 운세

4. Manse Calendar Section
   - Show four pillar cards in a grid:
     - 년주
     - 월주
     - 일주
     - 시주
   - Each card should feel like a small ink-stamped tile.
   - Use strong borders, cream background, dark plum text, and subtle gold accents.

5. Reading Cards
   - Long-form text cards for each result section.
   - Use letter-paper styling: inner shadow, soft paper color, strong outer border.
   - Section labels should be small, refined, and easy to scan.

6. Action Buttons
   - Buttons: 링크 복사, 공유하기, PDF로 저장, 이메일 발송
   - Buttons should not dominate the page.
   - Use dark plum primary button and ivory outlined secondary buttons.

## Style Notes

- The page should look premium enough for a paid report.
- Make it feel more like a private philosophy house report than a generic horoscope app.
- Keep Korean body text highly readable.
- Preserve the app's soft rounded design language, but make borders stronger and the night-sky mood deeper.
- The design should print well as PDF: avoid important white text on dark backgrounds inside long content sections.
